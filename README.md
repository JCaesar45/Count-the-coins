## Overview

This project is a **creative coin change calculator** built with HTML, CSS, and advanced JavaScript. Users can input an amount in cents, see the total number of ways to make change using US coins (quarters, dimes, nickels, pennies), and enjoy animated coin visuals that make the experience engaging and impressive.

---

## Features

- **Elegant & modern design** with gradient backgrounds and animated header.
- **Responsive UI** optimized for various devices.
- **Interactive input** with validation.
- **Dynamic programming** logic to accurately compute combinations.
- **Animated coins** that bounce and scatter for visual appeal.
- **Clean, well-structured code** suitable for showcasing skills during interviews.

---

## Technologies Used

- **HTML5** for structure
- **CSS3** for styling and animations
- **JavaScript ES6+** for logic and interactivity

---

## How to Use

1. **Clone or download** this repository.
2. Open `index.html` in your preferred web browser.
3. Enter an amount in cents (positive integer).
4. Click **"Calculate Ways"**.
5. View the total number of combinations and enjoy animated coins illustrating the calculation.

## Code Structure

- **HTML (`index.html`)**: Defines the webpage layout, including input, button, result display, and animation container.
- **CSS (`styles.css`)**: Styles the page with gradients, fonts, animations, and responsive design.
- **JavaScript (`script.js`)**: Implements the coin change calculation using dynamic programming, handles user interactions, input validation, and creates animated coin visuals.
